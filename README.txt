Graph partitioning API - Spring project installation
1. Create folder with name "saved" on your D:\ drive
2. Open console in the main project folder Graph_Partitioning_Api
3. Write the following command: mvn clean install
4. Open the project in Eclipse (Import it as Maven project)
5. Open project properties and find Deployment assembly. After that:
Add -> Java Build Path Entries -> Maven Dependencies-> Finish-> Apply and close
6. The project is setup. It can be started on TomCat server on http://localhost:8080/GraphPartitioning/
